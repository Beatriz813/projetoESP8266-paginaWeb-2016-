Informações Sobre o Projeto

Este projeto foi realizado nas dependências do Laboratório de Sistemas Microprocessados localizado no IFPA, sob a supervisão do professor Alcides Machado. O projeto tem como principal objetivo realizar o controle do Microcontrolador PIC18F4550 via WiFi, para isso foram utilizados: o módulo WiFi ESP8266; o MCU PIC18F4550; o Kit Acepic de Desenvolvimento, onde está alocado o PIC; e a Proto-Power-Supply, onde se encontra o ESP.

O intuito deste projeto é executar ações no Kit Acepic, através de comandos enviados por meio de uma rede wireless. A partir deste conceito, podem ser realizadas monitorias de qualquer lugar e a qualquer hora.

Desenvolvido por: Beatriz Alcântara.
